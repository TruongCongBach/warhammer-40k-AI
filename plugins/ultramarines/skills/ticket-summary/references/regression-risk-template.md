# Regression Risk Template

## Purpose

Use this template to summarize where the change could have unintended impact.

## Template

```markdown
Regression risks
- Shared logic impact: ...
- State or data risk: ...
- UI or interaction risk: ...
- Platform or environment risk: ...
```

## Common Risk Categories

- shared component behavior
- validation or transformation logic
- caching or stale state
- route or navigation transitions
- API request or response handling
- empty, loading, or error states
