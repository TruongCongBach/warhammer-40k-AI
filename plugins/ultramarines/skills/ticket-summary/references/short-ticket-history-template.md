# Short Ticket History Template

## Purpose

Use this template to create compact summaries for Jira comments or ticket history.

## Template

```markdown
Short summary for ticket history
- Fixed the issue by ...

Short summary for engineering readers
- Corrected ... by updating ... so that ...
```

## Rules

- Keep each summary to one or two lines.
- Mention the behavior change, not just the file change.
- Make the engineering summary slightly more technical than the ticket-history summary.
