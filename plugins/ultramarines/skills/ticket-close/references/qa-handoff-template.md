# QA Handoff Template

## Purpose

Use this template to create a concise QA handoff note.

## Template

```markdown
QA handoff note
- Primary flow to verify: ...
- States or edge cases to verify: ...
- Platform or environment notes: ...
```

## Guidance

- Name the most important validation path first.
- Keep this focused on practical verification, not engineering explanation.
- Omit platform or environment notes when they do not matter.
