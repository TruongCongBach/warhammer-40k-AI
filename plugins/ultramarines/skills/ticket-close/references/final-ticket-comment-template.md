# Final Ticket Comment Template

## Purpose

Use this template for the final Jira comment after approval.

## Template

```markdown
Final ticket comment
- Completed: ...
- What was fixed: ...
- QA focus: ...
- Risks / notes: ...
```

## Writing Rules

- Keep each line short and meaningful.
- Prefer behavior-level wording over file-level wording.
- Omit the risks line when there are no meaningful notes.
